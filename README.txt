# Synod System 
Turn a budget smartwatch (LJ755_MB) into a powerful PC Macro Deck.

Features
- **Movie Mode:** One-tap adjustment of brightness and volume.
- **Stealth Mode:** Instantly hide all windows.
- **Media Control:** Use watch swipes to control Spotify and Brave.
- **Emergency Kill:** Use the "Find Phone" Stop button to close any app.

Installation
1. Install Python 3.10+
2. Install dependencies:
   ```bash
   pip install bleak pyautogui pygetwindow CustomTkinter screen-brightness-control
